# Frontend - Modern E-Commerce Marketplace

The frontend of our full-stack e-commerce platform, built with cutting-edge technologies for a seamless and responsive user experience.

## 🎨 Features

- **React.js:** Modern React with Hooks for dynamic UI components
- **Vite:** Lightning-fast build tool for optimal development experience
- **Tailwind CSS:** Utility-first CSS framework for responsive design
- **Zustand:** Lightweight state management solution
- **Mobile-First Design:** Fully responsive layouts for all devices

## 📦 Technology Stack

- **React 18+** - UI framework
- **Vite** - Next generation build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Zustand** - State management
- **Axios** - HTTP client for API calls
- **ESLint** - Code quality enforcement

## 🚀 Getting Started

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📁 Project Structure

```
src/
├── components/        # Reusable React components
├── pages/            # Page components
├── stores/           # Zustand store management
├── lib/              # Utilities and API configuration
├── App.jsx           # Root component
└── main.jsx          # Entry point
```
